# 性能基准报告

## 1. 验证目标

- 注册接口平均耗时 `<= 300ms`
- 登录接口平均耗时 `<= 200ms`
- 数据库查询 `P99 <= 100ms`
- Serverless 冷启动数据库连接 `<= 1500ms`

## 2. 测试前置条件

- 已注入真实 `DATABASE_URL`、`DIRECT_DATABASE_URL`、`REDIS_URL`
- 已完成 Prisma 迁移和 seed
- 已接入真实 SendGrid / Sentry / Redis 服务
- 已在 staging 或 production-like 环境部署

## 3. 推荐执行命令

```bash
pnpm --filter @youda/database benchmark:serverless
pnpm --filter web test:coverage
pnpm --filter web build
```

压测工具建议：

- `k6`：用于注册、登录、资料读取的接口压测
- `wrk`：用于单接口吞吐验证
- Sentry Performance：用于 P95 / P99 趋势留档
