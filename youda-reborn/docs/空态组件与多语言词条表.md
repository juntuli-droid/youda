# 空态组件与多语言词条表

## 组件
- `apps/web/src/components/empty/EmptyStateCard.tsx`
- 支持中英双语、无障碍 `role="status"`、曝光/点击埋点。

## 已覆盖场景
- 未上传头像
- 未填写资料
- 无好友
- 无动态
- 无生涯
- 无消息
- 网络错误
- 权限受限
- 搜索为空

## 埋点字段
- `scenario`
- `action`
- `locale`
- `durationMs`
- 用户维度通过服务端从 session 补齐

## UI 走查结论
- 所有空态按钮点击区域保持 `>= 48px`
- 组件在浅色主题和资料页玻璃态容器中均可读
- 关键页面加载时优先展示 skeleton，再切换为空态，避免闪烁
