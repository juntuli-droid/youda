# 风控策略文档

## 限流
- 注册：`5 次 / 小时`
- 登录：`10 次 / 分钟`
- 发消息：`120 次 / 分钟`
- 匹配入队：`12 次 / 分钟`
- 所有限流统一返回 `429`，并带 `Retry-After` 头。

## 举报
- 接口：`POST /api/social/reports`
- 分类：`SEXUAL`、`HARASSMENT`、`FRAUD`、`INFRINGEMENT`、`OTHER`
- 审计字段：举报人、目标对象、IP、设备指纹、证据 URL、创建时间

## 拉黑
- 接口：`GET/POST/DELETE /api/social/blocks`
- 拉黑后好友列表会过滤该用户，私聊发送会被直接拒绝。
- 黑名单查询上限为 1000 条。

## 审计日志格式
- 统一写入 `audit_events`
- 关键字段：
  - `actorUserId`
  - `targetUserId`
  - `eventType`
  - `ipAddress`
  - `deviceFingerprint`
  - `payload`
  - `createdAt`
